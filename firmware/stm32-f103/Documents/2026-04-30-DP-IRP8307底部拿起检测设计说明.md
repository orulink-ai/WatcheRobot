# DP-IRP8307 底部拿起检测设计说明

本文记录 Watcher 底部拿起检测从旧红外数字中断模块切换到 DP-IRP8307 反射式红外器件后的硬件连接、CubeMX 配置和调试建议。

## 1. 器件结论

DP-IRP8307 不是 I2C、串口或带比较器输出的红外模块，而是一个贴片反射式光电器件，内部包含：

- 940 nm 红外发射 LED
- NPN 光电三极管

因此它本身不会直接输出稳定的数字 `INT` 信号。更推荐用 STM32 ADC 读取反射强度，再在软件里做阈值、防抖和滤波判断。

## 2. 引脚定义

根据 DP-IRP8307 数据手册：

| 焊盘 | 功能 | 建议连接 |
| --- | --- | --- |
| 1 | Anode，红外 LED 正极 | 通过限流电阻接发射侧电源 |
| 2 | Cathode，红外 LED 负极 | GND |
| 3 | Emitter，光电三极管发射极 | GND |
| 4 | Collector，光电三极管集电极 | `BOTTOM_IR_ADC` |

注意：ADC 采样点接的是 `Pin4 Collector`，不是 `Pin3 Emitter`。

## 3. 推荐原理图连接

发射侧：

```text
3V3 -> R_LED -> DP-IRP8307 Pin1 Anode
DP-IRP8307 Pin2 Cathode -> GND
```

接收侧：

```text
WATCHER_3V3
 |
R_PULLUP
 |
BOTTOM_IR_ADC ---- STM32 PB0 / ADC1_IN8
 |
C_FILTER
 |
GND

BOTTOM_IR_ADC ---- DP-IRP8307 Pin4 Collector
DP-IRP8307 Pin3 Emitter ---- GND
```

推荐初始参数：

| 器件 | 初始值 | 说明 |
| --- | --- | --- |
| `R_LED` | 220R 或 330R | 3.3 V 发射侧限流 |
| `R_PULLUP` | 10k | 将光电三极管电流转换成 ADC 电压 |
| `C_FILTER` | 10 nF 到 100 nF | ADC 点低通滤波，100 nF 对拿起检测也可用 |

`R_PULLUP` 不是 ADC 分压电阻，而是必须的电流-电压转换电阻。没有它时 ADC 节点会悬空，读数不稳定。

## 4. ADC 电压逻辑

使用上拉电阻和光电三极管下拉时，ADC 逻辑是反相的：

```text
反射弱 / 拿起 / 远离桌面  -> 光电三极管导通弱 -> ADC 电压高
反射强 / 靠近桌面         -> 光电三极管导通强 -> ADC 电压低
```

软件判断时不要直接假设“ADC 越高代表越近”。实际应通过串口打印或日志采样，分别记录：

- 空中拿起时 ADC 平均值
- 放在目标桌面上时 ADC 平均值
- 不同桌面材质和外壳开孔高度下的边界值

最终阈值建议放在两组稳定均值之间，并增加迟滞和时间防抖。

## 5. CubeMX 配置建议

新板建议使用：

```text
PB0 = ADC1_IN8
User Label = BOTTOM_IR_ADC
```

ADC1 Regular Conversion 建议：

如果保留当前舵机反馈 ADC：

```text
Number Of Conversion = 3
Rank 1 = Channel 0    // SERVO1_FB
Rank 2 = Channel 5    // SERVO2_FB
Rank 3 = Channel 8    // BOTTOM_IR_ADC
Sampling Time = 239.5 Cycles
```

如果只单独调试 DP-IRP8307：

```text
Number Of Conversion = 1
Rank 1 = Channel 8
Sampling Time = 239.5 Cycles
```

其他建议保持：

```text
Scan Conversion Mode = Enabled      // 多通道时启用
Continuous Conversion Mode = Disabled
External Trigger Conversion Source = Software Start
Data Alignment = Right alignment
```

旧的 `PB5 / IR_PROX_INT` 原本适合接带比较器的数字红外模块。切换到 DP-IRP8307 裸器件后，不建议继续把它当唯一输入，除非外部增加比较器，把模拟反射强度先转换成数字电平。

## 6. 5 V 发射侧兼容性

DP-IRP8307 数据手册给出的红外 LED 关键参数：

```text
IF max = 20 mA
VF typ = 1.4 V
VF max = 1.6 V
```

红外 LED 发射侧可以使用 5 V，但必须重新选择限流电阻：

```text
5V -> 390R/470R -> Pin1 Anode
Pin2 Cathode -> GND
```

不建议在 5 V 下继续使用 220R 长期开启，电流约为：

```text
(5 V - 1.4 V) / 220R ~= 16 mA
```

虽然低于 20 mA 绝对最大值，但对电池设备偏大。

无论发射侧是否使用 5 V，接收侧 ADC 上拉都必须使用 3.3 V：

```text
3V3 -> R_PULLUP -> BOTTOM_IR_ADC -> STM32 ADC
```

不要把 `BOTTOM_IR_ADC` 上拉到 5 V，否则可能损坏 STM32 ADC 输入。5 V 和 3.3 V 电源域必须共地。

## 7. 后续省电优化

打样阶段可以让红外 LED 常亮，方便验证 ADC 值。稳定后建议增加一个 `IR_LED_EN` 控制脚，通过 NMOS 控制红外 LED 电流。

推荐采样流程：

```text
1. 关闭红外 LED，读取 ADC，得到环境光基线
2. 打开红外 LED，等待模拟节点稳定
3. 再次读取 ADC，得到环境光 + 反射光
4. 用两次读数差值判断底部反射状态
5. 关闭红外 LED
```

这样可以降低功耗，也能减小环境光变化对拿起检测的影响。

## 8. 打样检查清单

- `Pin1` 是否通过限流电阻接到发射侧电源
- `Pin2` 是否接 GND
- `Pin3 Emitter` 是否接 GND
- `Pin4 Collector` 是否接 `BOTTOM_IR_ADC`
- `BOTTOM_IR_ADC` 是否通过 10k 左右上拉到 3.3 V
- `BOTTOM_IR_ADC` 是否没有任何 5 V 上拉
- CubeMX 是否把 `PB0` 配为 `ADC1_IN8`
- ADC regular rank 是否实际包含 `Channel 8`
- 串口或日志是否能打印拿起/放下两种状态下的 ADC 原始值
